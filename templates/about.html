Fusing advanced Natural Language Processing (NLP) with the dynamic capabilities of API keys, RoboWhiz delivers engaging and personalized responses in real-time. It is build on the following Problem Statement

The Employment Department at present has a digital platform www.pgrkam.com and its mobile application to provide almost all services offered to job seekers and employers through digital means. The portal has multiple modules like private sector jobs, government jobs, selfemployment avenues, foreign jobs, foreign study, counseling, guidance, induction into armed forces, job melas, etc. Currently, when a user visits the portal/app, there is no hand-holding mechanism to help the user get to the part of the digital platform which will resolve his queries. The user is required to navigate across multiple modules on the portal/ app to search for answers.

The idea is quite simple. We will create a database or if we get access to the official webiste database we will train this bot using NLP and different API
Then we will use javascript to fetch data from database and present it to the user through the chatbot.

• The chatbot able to intelligently addresses text an voice queries in English /Hindi /Punjabi around job searches . A multilingual screen module also added for better query handling.
• The app also able to maintain candidate’s history for better personalised recommendations.
• This bot can recommend job courses, according to users qualifications and it can predict your job possibility through flowchart.

This RoboWhiz Chatbot is a demo itself as it is running on a Flask server. We have integrated Open AI's API key to generate response.
In the Flask app using SQLAlchemy we have created a intent database & an Database named page where the intents will be added in the database using flask.
In the javascript file we are currently creating logic to fetch data from the database and present it to the user.
